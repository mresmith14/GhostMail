# Temp Mail Inserter

Temp Mail Inserter is a lightweight Chrome extension that lets you quickly insert a disposable email address into any web form. Perfect for signing up for websites without giving away your real email, keeping your inbox clean, and protecting your privacy.

---

## Features

- ✅ One-click insertion of a temporary email
- ✅ Works on any web form
- ✅ Uses free temporary email services (no account required)
- ✅ Privacy-focused: no data is stored or sent from your browser

---

## How to Use

1. Install the extension in Chrome.
2. Navigate to any sign-up form.
3. Click the Temp Mail Inserter icon in the toolbar.
4. Your temporary email is automatically inserted into the active email field.

## Installation

1. Download the repository or clone it:

```bash
git clone https://github.com/YOUR-USERNAME/temp-mail-inserter.git
````

2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode**.
4. Click **Load unpacked** and select the cloned repository folder.
